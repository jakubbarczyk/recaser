# Recaser

**Bulk-rename files to match a certain case.**

## Installation

Installing the package with _npm_:

```shell
npm install --global recaser
```

Installing the package with _yarn_:

```shell
yarn global add recaser
```

## Usage

Recaser handles the following cases:

- lower
- upper
- camel
- pascal
- kebab
- snake
- train
